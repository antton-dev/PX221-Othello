# Documentation technique 
## Récuperer le dépot (initialisation)
	user@machine:~/Documents/$ git clone https://github.com/antton-dev/PX221-Othello.git
La racine du projet est donc `/Documents/PX221-Othello/`
## Workflow de modification
1. Récuperer la dernière version distante : `git pull`
2. Faire les modifs du code
3. Ajouter les modif
 
		git add .
		git commit -m "description"
		git push -u origin master
